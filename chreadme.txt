# ChatBot AI Assistant

Assistente virtual inteligente para atendimento ao cliente, treinado para responder perguntas frequentes e resolver dúvidas comuns.

## Tecnologias Utilizadas
- Python
- TensorFlow
- Flask
- Natural Language Processing (NLP)

## Funcionalidades
- Chatbot responsivo
- Aprendizado contínuo com Machine Learning
- Atendimento a múltiplos usuários simultaneamente
- Integração com WhatsApp e WebChat
